# Generative Portfolio

A single-page, design-first portfolio website showcasing projects and use cases developed with the assistance of AI/LLMs.

## Tech Stack

- **Core Framework**: Astro
- **Styling**: HTML5 / CSS3 (Vanilla CSS with CSS custom variables, modern grid layouts, and flexbox)
- **Logic & Animation**: JavaScript / GSAP (GreenSock Animation Platform for smooth scroll-driven triggers)
- **Icons**: Lucide Icons (lucide-static)
